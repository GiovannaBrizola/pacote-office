O Microsoft Excel é um editor de planilhas produzido pela Microsoft para computadores que utilizam o sistema operacional Microsoft Windows, além de computadores Macintosh da Apple Inc. e dispositivos móveis como o Windows Phone, Android ou o iOS. Seus recursos incluem uma interface intuitiva e capacitadas ferramentas de cálculo e de construção de tabelas que, juntamente com marketing agressivo, tornaram o Excel um dos mais populares aplicativos de computador até hoje.

Atalhos Funções
Ctrl + N Para criar uma nova pasta de trabalho
Ctrl + S salvar uma pasta de trabalho
Ctrl + A selecionar todo o conteúdo de uma pasta de trabalho
Ctrl + C Para copiar as células realçadas
Crtl+D Copia e cola o conteúdo da célula de cima
Ctrl + F Para pesquisar qualquer coisa em uma pasta de trabalho.
Ctrl + H Para localizar e substituir o conteúdo da célula
Shift+Barra de espaço Seleciona a linha inteira
Ctrl+Barra de espaço Seleciona a coluna inteira
Shift+Setas Seleciona as células, uma por uma
